## Setup/Installation Requirements

1\. Clone the repository:

```
git clone https://github.com/hwisoo/react-portfolio.git
```

2\. Navigate into project directory

3\. Install the dependencies:

```bash
npm install
```

4\. Run the Program:

```bash
npm run start
```
